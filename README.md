Flask Todo App 📝

A simple CRUD (Create, Read, Update, Delete) web application built using Flask, SQLAlchemy, SQLite, Bootstrap, HTML, and CSS.

This project was created as part of my learning journey in Flask web development and helped me understand how frontend, backend, and databases work together in a real application.

🚀 Features

- Add new tasks
- View all tasks
- Update existing tasks
- Delete tasks with confirmation popup
- Search tasks
- Flash notifications for actions
- Responsive Bootstrap user interface
- Dynamic routing with Flask
- SQLite database integration

🛠️ Technologies Used

- Python
- Flask
- SQLAlchemy
- SQLite
- Bootstrap
- HTML
- CSS
- Jinja2 Templates

📚 Concepts Learned

- Responsive UI Design with Bootstrap
